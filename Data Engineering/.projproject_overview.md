Project Overview

This project presents the design of a Voltage Threshold Indicator using the LM393 Comparator. The main objective of the circuit is to monitor an input voltage and indicate when the voltage crosses a predefined threshold level. The LM393 comparator compares the input voltage with a reference voltage set using a voltage divider and potentiometer.

When the input voltage exceeds the preset threshold value, the comparator output changes state and activates an LED indicator. This helps in easily detecting whether the voltage level is above or below the specified limit. The circuit is simple, reliable, and useful for voltage monitoring applications.

The project was designed using KiCad for schematic and PCB layout development. The design includes components such as the LM393 comparator IC, resistors, potentiometer, LEDs, and a power supply connector. This system can be used in applications such as battery monitoring, power supply protection, and electronic voltage detection systems.
